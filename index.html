ranPic

It is a simple application for generating random images. You can set the size, the mode(black and white or colorful), and the mirror. Using the mirror you will get symmetrical images. My favourite settings are: black and white, mirror +, 8x8 or colorful, no mirror, 2x2.

The app has another main functionality which is the input part. You can code with one-digit numbers, with each meaning a color. The app can be used with kids to understand the concept of the computers, by coding with 0s and 1s and getting back some visual information.

You can try it out online or you can download the Android app.

BrainFreak

It is all about a programming language, called BF or Brainfuck.

The site includes a tutorial, an interpreter and an editor for this esoteric language. It lists some links, and a list of pieces of codes that make something interesting.
